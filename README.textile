h1. iPhone Font Browser (Objective-C, 2008)

*Version:* 1.0<br/>
*Author:* Adrian Kosmaczewski<br/>
*Date:* November 18th, 2008<br/>
*Programming Languages:* Objective-C<br/>
*Tools:* Xcode<br/>
*Platforms:* Mac OS X 10.5 “Leopard” (Intel only)<br/>
*Repository:* "Github":http://github.com/akosma/fontbrowser/<br/>
*License:* None, public domain, whatever you want :)
